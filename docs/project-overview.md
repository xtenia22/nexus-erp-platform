# Nexus ERP Platform

## Objetivo

Construir una plataforma web moderna conectada a un sistema empresarial mediante API, con el fin de aprender y aplicar buenas prácticas actuales de desarrollo full stack.

## Alcance inicial

El proyecto comenzará con un MVP orientado a catálogo web de productos, consumiendo datos desde una capa de integración con el sistema empresarial.

## Objetivos de aprendizaje

- Organización de un proyecto full stack moderno
- Frontend con Next.js y TypeScript
- Backend con NestJS y TypeScript
- Consumo de APIs
- Arquitectura por capas
- Uso de tickets, backlog y sprints
- Buenas prácticas de versionado con Git y GitHub
